# Traceloom Minimal Requirement Workspace

This directory is the default starter template for one narrow Traceloom requirement slice.
Start with `01_brief.md`, then grow the slice into `PRD` and `solution_design` only when the current stage is ready.

## Included files

- `01_brief.md`

## Recommended first edits

- replace `starter-feature` with a real feature key
- update the artifact ID to match your date/version scheme
- keep the first slice narrow enough for one PM and one tech lead to review quickly

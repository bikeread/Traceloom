# Traceloom Starter Repo

This directory is a minimal starter template for a six-artifact Traceloom feature slice.
Copy it into a new repository or a sandbox folder, then replace placeholder IDs, titles, and scope fields.

## Included files

- `01_brief.md`
- `02_prd.md`
- `03_solution_design.md`
- `04_execution_plan.md`
- `05_test_acceptance.md`
- `06_release_review.md`

## Recommended first edits

- replace `starter-feature` with a real feature key
- update artifact IDs to match your date/version scheme
- narrow the feature so one team can review the full chain quickly
